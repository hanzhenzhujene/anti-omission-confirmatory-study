# Scenarios

This repository keeps only two scenario banks.

## `confirmatory_holdout_v3/`

The final locked confirmatory bank used for the paper’s main evidence.

- `24` risk scenarios
- `36` benign controls
- used in `configs/experiment/mainline_confirmatory_holdout_v3_live.json`

## `dev/`

A tiny development-only bank retained exclusively for smoke testing and code-path verification.

It is not confirmatory evidence and should not be cited as part of the paper’s empirical results.
